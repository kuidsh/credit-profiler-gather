Prioridad de decisión (peso descendente)
# Variable ¿Por qué importa?
1 Historial crediticio + comprobación de ingresos Mayor peso de todos. Historial bueno + comprobación sólida → ruta tradicional. Malo o sin comprobación → alternativa o reestructura.
2 Ocupación + antigüedad laboral Asalariado o pensionado abren ruta tradicional. Independiente/negocio propio dependen de comprobación. Informal tiende a alternativa.
3 Relación de carga financiera (%) Si el % supera 40% la operación se vuelve apretada. Si supera mucho ese umbral, se sugiere reestructurar antes de ingresar.
4 Enganche disponible A mayor enganche, mejor lectura. En perfiles frágiles el enganche es la palanca más importante para mejorar la viabilidad.
5 Año modelo del auto Autos más recientes abren más rutas. Modelos muy antiguos complican o reducen las opciones disponibles.
6 Congruencia mensualidad vs ingreso Si la mensualidad buscada es poco congruente con el ingreso, el perfilador sugiere ajustar antes de presentar.
Fórmula de carga financiera
Cálculo orientativo
Carga % = (Deudas mensuales + Mensualidad buscada) ÷ Ingreso mensual × 100
✅ Cómoda
≤ 35% — operación sana, cliente con margen
⚠️ Justa
36%–40% — aceptable, requiere cuidado
🚨 Apretada
> 40% — riesgosa, recomendar reestructura
Clasificación de viabilidad inicial
Alta
Perfil: Tradicional

El caso luce bien perfilado. Cliente formal, ingreso estable, historial bueno y enganche razonable. Apto para rutas competitivas directamente.

Media
Perfil: Flexible / Tradicional con ajustes

El caso puede funcionar, pero hay puntos de tensión: comprobación parcial, historial regular o carga ajustada. Requiere ajustes o cautela.

Baja
Perfil: Delicado / Alternativo

El caso luce delicado: historial malo, sin comprobación, carga apretada o enganche muy bajo. Conviene reestructurar antes de moverlo.

Reglas de negocio por variable

📋 Historial crediticio

  • Bueno → favorece ruta tradicional
  • Regular → puede requerir ajustes
  • Malo → alternativa o reestructura
  • Sin historial → no elimina viabilidad, requiere cuidado

📄 Comprobación de ingresos

  • Sí comprueba → ruta tradicional
  • Parcial → mueve el caso a flexible
  • No comprueba → reduce opciones tradicionales

👷 Ocupación

  • Asalariado / pensionado → mejor apertura en tradicional
  • Independiente / negocio propio → viable, depende de comprobación
  • Informal → ruta alternativa o ajustes

💰 Enganche

  • Mayor enganche → mejor lectura general
  • Enganche bajo → debilita la operación
  • En perfiles frágiles → palanca más importante

🚗 Año modelo del auto

  • Reciente → abre más rutas de financiamiento
  • Antiguo → puede complicar las opciones
  • Unidades especiales → mayor cautela

📊 Capacidad de pago

  • Carga alta → el perfilador advierte al vendedor
  • Mensualidad incongruente → sugiere ajuste
  • Aunque el caso sea movible, si la carga es apretada → se señala
