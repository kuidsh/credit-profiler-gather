The user is building a commercial automotive credit profiling tool for Mexican car dealership salespeople. The project is a multi-agent collaboration:

- This agent (wizard-frontend-builder) owns: Step3, Step4, usePerfilador hook, promptBuilder, responseParser
- Another agent owns: WizardContext, shared UI components, Step1, Step2, App.jsx, config files
- The user coordinates both agents

Target end-user: car floor salespeople using a mobile phone while standing at the dealership. The UX must be fast (1–3 minutes), large touch targets (min 44px), Spanish-only, no banking jargon.
